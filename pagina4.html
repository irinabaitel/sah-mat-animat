Ce este șahul dublu?

Șahul dublu este o situație specială în care regele advers este atacat simultan de două piese.

Spre deosebire de un șah obișnuit, acest tip de atac este extrem de puternic, deoarece singura opțiune de apărare este mutarea regelui — capturarea sau blocarea ambelor atacuri fiind imposibilă.

Șahul dublu apare prin descoperire: o piesă se mișcă și dezvăluie atacul unei alte piese, iar în același timp se așează pe un câmp de unde atacă și ea regele.

🔥 De ce este șahul dublu atât de puternic?
Adversarul nu poate scăpa decât mutând regele — nu există altă formă de apărare! Aceasta face din șahul dublu o armă tactică extrem de periculoasă.

Exemplul 1 — Cal + Turn

Cal + Turn:
Apasă pentru a vedea șahul dublu!

Exemplul 2 — Cal + Nebun

Cal + Nebun:
Apasă pentru a vedea șahul dublu!

Exemplul 3 — Nebun + Turn

Nebun + Turn:
Apasă pentru a vedea șahul dublu!
